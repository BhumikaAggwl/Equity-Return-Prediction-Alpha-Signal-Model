# Equity-Return-Prediction-Alpha-Signal-Model
ML pipeline predicting 21-day excess stock returns vs S&amp;P 500 across 100 equities. Compares Decision Tree, XGBoost (scratch + library), LightGBM, CatBoost via walk-forward validation and Information Coefficient. Generates BUY/HOLD/SELL signals with SHAP explainability and FinBERT sentiment.
